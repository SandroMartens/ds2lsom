DS2L-SOM
========

**DS2L-SOM** is a topological, density-based clustering algorithm that combines a
Self-Organizing Map (SOM) for prototype learning with Gaussian KDE density
estimation and gradient ascent to detect clusters — without requiring the number
of clusters to be specified in advance.

Based on the papers by Cabanes, Bennani and Fresneau (2008, 2012).
